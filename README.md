# Анализ рынка квартир в Москве

---

## Описание проекта

Проект направлен на **исследование рынка недвижимости Москвы** с использованием реальных данных о квартирах.  
**Цель** — выявить ключевые факторы, влияющие на **цену квартиры**, и подготовить чистый, структурированный датасет для статистического анализа и машинного обучения.

Проект состоит из **трёх последовательных этапов**:

1. **Разведочный анализ (EDA)**  
2. **Очистка и предобработка данных**  
3. **Проверка статистических гипотез**

## Этапы анализа

---

### 01_EDA — Разведочный анализ данных

**Цель:** Получить общее представление о данных, выявить аномалии и закономерности.

**Ключевые выводы:**
- Нет пропусков и дубликатов
- Распределения **не нормальные** → использовать **медиану, IQR, непараметрические тесты**
- **Мультимодальность** — типовые планировки, классы жилья, зональность города
- **Сильная корреляция цены с общей площадью** (`r = 0.769`)

| Признак   | Медиана     | Особенности |
|-----------|-------------|-------------|
| `price`   | 115 млн     | Правый хвост (элитные объекты) |
| `totsp`   | 73.5 м²     | Биомодальное |
| `livesp`  | 45 м²       | Мультимодальное |
| `kitsp`   | 9 м²        | Пики: 6–7, 8–9, 10–11 м² |
| `dist`    | 12 км       | Зональность города |

---

### 02_Data_Cleaning — Очистка и предобработка

**Цель:** Подготовить чистый, корректный датасет.

**Выполнено:**
- Удалены лишние колонки: `Unnamed: 0`, `code`
- Созданы новые признаки:
  - `close_to_center` — пешая доступность до центра
  - `room_type` — 1-, 2-, 3-, 4+ комнатные
  - `price_per_m_2` — цена за кв. м
- Бинарные признаки → `Yes/No`
- Удалены выбросы:
  - Логические (кухня > 30 м², жилая > общей)
  - IQR (k=3) по площадям
  - Глобальные и **локальные по группам `room_type`** по цене

**Результат:**
- **1890 строк** (из 2040)
- Все распределения **асимметричны** → **непараметрические тесты**

---

### 03_Hypothesis_Testing — Проверка гипотез

**Все нулевые гипотезы отвергнуты (p < 0.05)**

| Признак                     | Тест                     | Эффект на цену |
|----------------------------|--------------------------|----------------|
| `total_area`               | Spearman                 | **+0.769** |
| `close_to_center` = Yes    | Mann-Whitney             | **+20.5%** |
| `room_type`                | Kruskal-Wallis           | **+15–20 млн/комната** |
| `is_brick` = Yes           | Mann-Whitney             | **+15.9%** |
| `is_middle_floor` = Yes    | Mann-Whitney             | **+9.5%** |
| `walkable_to_metro` = Yes  | Mann-Whitney             | **+6.5%** |

| Ранг | Фактор                          | Влияние |
